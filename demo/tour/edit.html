Edit Content

Now that we have created a new post and used getContent() to store that post on the server, let’s look at how we can reload and edit that post back into TinyMCE.

First, click on the Edit button to the right of the post you created in Step 1. This loads the same form, TinyMCE and the saved content into the browser.

/src/xxx/xxx.js contains the function loadContent(). This function is responsible for getting your existing post content from our server and loading it into the form. TinyMCE’s setContent() API loads the content into TinyMCE.

As with getContent(), you do have to target a specific editor instance but once you have an editor instance targeted you can use setContent() to load an existing string of HTML.
