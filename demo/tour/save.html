Save

Clicking the Save button extracts the content from each field in the form and sends it to the server by calling the /save API running on the Node server. Only the content and not the actual HTML form is sent to the server. Examining /src/xxx/xxx.js you can see that this is done via AJAX. Use the getContent() API to get data from the editor. This API gets the current HTML content from the editor instance. We combine that data with the data from the standard input field and send that to the server.

For more information on getting content from the editor, refer to getContent().

NOTE: Although there is only one editor instance in this example application, there are multiple ways to target a specific editor instance on a page. Use the get() method to identify which editor instance we want to target.
