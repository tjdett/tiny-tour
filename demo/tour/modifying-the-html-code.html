Developer/Power User Spotlight: Modifying the HTML code

You can manually modify the HTML code directly without using TinyMCE’s default WYSISYG mode. The Source code toolbar button () opens the Advanced Code Editor.

The Advanced Code Editor brings a familiar IDE-like code editor to TinyMCE making it simpler to modify the HTML.

  • color syntax highlighting
  • bracket matching
  • code folding
  • multiple selections and carets

Open the Advanced Code Editor and make some changes to the HTML. Click OK to make your changes visible in the editor’s WYSIWYG view.
