Create Content

The first screen that loads is comprised of two fields that allows you to create a new post.

  1. The Title field is a simple input field.
  2. The Content field is managed by TinyMCE.

The /src/xxx/xxx.js file contains both an input and a textarea fields. Calling tinymce.init() causes TinyMCE to appear. Configuration of TinyMCE is done by passing a simple JavaScript object.

The selector property is the only required element in the configuration object. This configuration option tells TinyMCE which element(s) on the page to replace with TinyMCE.

Enter content into the Title and Content fields. Click the save button to create new pieces of content.
