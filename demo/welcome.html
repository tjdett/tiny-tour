Welcome to the Tiny Tour

Who should use this?

To use this quickstart of TinyMCE you need a working understanding of the following web technologies:

  • HTML
  • CSS
  • JavaScript

Understanding the following technologies and concepts is helpful but not required for this quickstart:

  • Node / NPM
  • REST APIs

Learning Objectives

This micro-blog application will teach you:

  • Create and edit content via a web browser.
  • Extract and save content to a data store via a REST API.
